# Sales Forecasting and Inventory Optimization

## Project Overview
This project focuses on sales forecasting and inventory optimization using retail/e-commerce datasets. The aim is to implement and compare different forecasting models and analyse whether any specific model outperforms others. The best forecasting results will be used to support inventory-related decisions such as stock planning, reorder point, safety stock, and reducing stockouts or overstocking.

## Problem Statement
Retail and e-commerce businesses need accurate sales forecasting to manage inventory effectively. Incorrect forecasts can lead to stockouts, lost sales, overstocking, increased storage costs, and reduced profitability.

## Objectives
- Review relevant research papers on sales forecasting and inventory optimization.
- Identify and evaluate suitable public datasets.
- Perform data cleaning, preprocessing, and exploratory data analysis.
- Implement baseline, statistical, machine learning, and deep learning forecasting models.
- Compare model performance using suitable evaluation metrics.
- Connect forecasting results with inventory optimization decisions.

## Datasets Considered
- Walmart Recruiting: Store Sales Forecasting
- Store Item Demand Forecasting Challenge
- Corporación Favorita Grocery Sales Forecasting
